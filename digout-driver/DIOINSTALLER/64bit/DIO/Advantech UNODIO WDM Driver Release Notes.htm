Top of page

Overview

Advantech UNODIO WDM Driver contains a set of functions and associated structures that can be used in various application programs for interfacing with Driver. Examples of BCB, Console, VB.NET, C#.NET, VB are supplied in the package, providing a reference for users to develop applications.

System  Requirements 

  • Supported Operating Systems: Windows 7(32-bit and 64-bit);Windows Vista(32-bit and 64-bit); Windows XP(32-bit and 64-bit); Windows 2003(32-bit and 64-bit);Windows 2000
